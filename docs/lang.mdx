# Lang.jsonc - Complete Guide

## What is lang.jsonc?

This file contains all the **text labels** used throughout your website. If you want to change any text that appears on buttons, titles, or messages, you do it here.

**File Location:** `settings/lang.jsonc`

---

## Structure

```json
{
  "settings": {
    "inLanguage": "en"
  },
  "data": {
    "lightDark": "Light / Dark Theme",
    "cardGoUrl": "View",
    "viewMoreProjects": "Explore More Projects",
    ...
  }
}
```

---

## Language Setting

```json
"settings": {
  "inLanguage": "en"
}
```

This sets your website's language tag. Common codes:

| Code | Language |
|------|----------|
| `en` | English |
| `tr` | Turkish |
| `de` | German |
| `fr` | French |
| `es` | Spanish |
| `ar` | Arabic |

You can find more codes at: https://r12a.github.io/app-subtags

## How to Change Text

### Example 1: Change Button Text

Original:
```json
"cardGoUrl": "View",
```

Change to:
```json
"cardGoUrl": "See Details",
```

### Example 2: Change Section Titles

Original:
```json
"ourProjectsTitle": "Our Projects",
```

For an architecture firm:
```json
"ourProjectsTitle": "Our Portfolio",
```

For a construction company:
```json
"ourProjectsTitle": "Completed Projects",
```

### Example 3: Change Error Messages

Original:
```json
"notOurProjects": "No projects found!",
```

Change to:
```json
"notOurProjects": "Check back soon for our latest work!",
```

## Translation Guide

If you want to translate your website to another language:

### Step 1: Change Language Code

```json
"settings": {
  "inLanguage": "tr"  // Change to Turkish
}
```

### Step 2: Translate All Labels

```json
"data": {
  "lightDark": "Açık / Koyu Tema",
  "cardGoUrl": "Görüntüle",
  "viewMoreProjects": "Daha Fazla Proje Gör",
  "addressTitle": "Adres",
  "workHours": "Çalışma Saatleri",
  "servicesTitle": "Hizmetlerimiz",
  "ourProjectsTitle": "Projelerimiz"
}
```

## Pro Tips

### Tip 1: Keep Messages Friendly
Instead of "No projects found!", use "Coming soon!" or "Check back later!"

### Tip 2: Match Your Industry
- Architecture: Use "Portfolio", "Completed Works", "Designs"
- Construction: Use "Projects", "Built Structures", "Developments"
- Real Estate: Use "Properties", "Listings", "Available Units"
- Interior Design: Use "Designs", "Spaces", "Interiors"
- Agency: Use "Campaigns", "Case Studies", "Work"
- Portfolio: Use "Works", "Creations", "Projects"

### Tip 3: Test After Changes
Always refresh your website after changing lang.jsonc to make sure text displays correctly.

## Quick Checklist

After editing lang.jsonc, check:

- [ ] All text fits in buttons (not too long)
- [ ] Special characters display correctly (ç, ğ, ş, etc.)
- [ ] Text matches your brand voice
- [ ] No empty strings ("")
