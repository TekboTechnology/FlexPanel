# Documentation

Welcome to the documentation folder. This contains all the guides you need to configure your website.

## Quick Start

Go to **[docs.page/index.mdx](/docs.page/index.mdx)** to begin!

## What's Inside?

| File | What It Teaches |
|------|----------------|
| `docs.page/getting-started.mdx` | How to set up your website from scratch |
| `docs.page/config.mdx` | Website colors, logo, and appearance |
| `docs.page/pages.mdx` | Creating categories and pages |
| `docs.page/projects.mdx` | Adding your work samples |
| `docs.page/lang.mdx` | Translating text and labels |
| `docs.page/file-structure.mdx` | Where to put your files |
| `docs.page/index.mdx` | Main documentation page |

## How to Use

1. Open `docs.page/index.mdx` in your browser
2. Follow the guides step by step
3. Make changes to your JSON files in `/settings/`
4. Upload images to `/assets/media/img/`

## Need Help?

Start with the Getting Started guide - it explains everything in simple terms!
